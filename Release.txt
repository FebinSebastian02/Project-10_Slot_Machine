v1.0.0

Slot Machine is a program with which User can place bets and earn rewards by trying their luck was created with:-
	- functions which were defined and called.
	- global constants which were defined and used.
	- while loop that was used to run the program continuously until the user runs out of cash.
	- if else conditions that were used to take actions based on user's choice.
	- dictionaries that were used to store the symbols and symbol values in a slot machine reel.
	- list that was used to store reels of different columns.
	- random module that was imported to use the choice function to randomly choose a value from a list.
	- slice operator which was used to copy a list and Splat/Unpack operator which was used to fetch all values from a list.
	- for loops that were used to iterate through rows, columns etc.
